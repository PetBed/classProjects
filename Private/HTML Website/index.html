<!DOCTYPE html> <DOCTYPE>: Defines version of html

<html lang="en"> <html>: Root of an html document

<head> <head>: Contains page information

  <meta charset="UTF-8"> <meta>: Define meta data (used to specify data such as keywords, author, and viewport).
More information here.
  <meta name="viewport" content="width=device-width, initial-scale=1.0">

  <link rel="stylesheet" href="styles.css"> <link>: Defines relationship between the current document and external document
Usually used for linking external stylesheets or fonts. More information here.

  <title>Document</title> <title>: Defines document title (shown in top browser bar)

  <style> <style>: Define style information (also known as internal css)

  </style>

</head>

<body> <body>: Represents all page content

  <script src="script.js"></script> <script>: Defines Javascript resources (script can either go in between the tags or linked externally via src attribute)

</body>

</html>

02. Comments

Anything text that are encased in <!-- [Text] --> will be a comment. !If you're using Visual Studio Code, type ! then click tab to automatically generate page structure
(This is done using emmet. For more information click here) Any content in comments won't be run.
Comments are used future readablity/explanation or annotation.

Comments can be inline,

<p>Hello World!</p><!-- This is a comment -->

Single line,

<!-- This is a single line comment -->

Or multilined.

<!-- This is a...
...multiline comment -->
